Othello Project:
----------------




Step 1. Run the program and you'll be presented with a board, some instructions above and the option to enter a row number.

Step 2. Read the instructions if necessary, but then enter a a number for the row. 

Step 3. Enter a number for the column and your piece, if the move is valid, will be placed on the board.

Step 4. Repeat steps 2 and 3 for player 2. Play repeats between players until the board is occupied fully or someone wins.
